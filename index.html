01 — About

The Studio & The Maker

B-Games

Our mission is to create fun and innovative games that inspire people across the world. We put passion into programming, developing games and software that bring imagination to life.

B-Games is a sole proprietorship founded by Brent Maxwell Goldstein. Projects and game jams are viewable on GitHub and Itch.io.

Hi, I'm Brent

I created B-Games in September 2019 during my junior year of high school and have been building ever since. I'm a machine learning engineer and data scientist with a strong foundation in algorithms, data structures, and software engineering.

At ADP I work as an Associate ML Engineer and Data Scientist, where I've shipped products, created ML deployment pipelines, developed dashboards, and made internal tooling used by hundreds of engineers. Before that I interned at Warner Bros. Games Central Game Technology working on automated unit testing and frameworks. I've also worked for Amplifier Game Invest as a gameplay programmer and game designer, and I've competed in numerous game jams through the RPI Game Jam Club.

Some things I've worked on recently: cutting Onboarding onto the ADP AI Agent Platform, RAG retrieval latency, migrating 300+ models into Unity Catalog, building ADP's AI Playground dashboard, and leading the team that delivered ADP's first secure RAG system with PII scanning and full session history.
